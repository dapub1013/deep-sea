# Deep-Sea

A modern desktop music player for streaming Phish concerts from phish.in.

## Description

This project develops a polished application for streaming Phish concerts, serving as a learning exercise to improve development skills and create a reusable framework.

## Features

- Stream concerts from phish.in API
- Browse shows, setlists, venues, and tours
- Core playback features: play/pause, track navigation, volume control
- Cross-platform support: macOS and Raspberry Pi

## Technologies

- React with Vite
- Tailwind CSS
- shadcn/ui components

## Getting Started

The UI prototype is located in `docs/04-ui-ux-design/`.

To run the prototype:

```bash
cd docs/04-ui-ux-design
npm install
npm run dev
```
